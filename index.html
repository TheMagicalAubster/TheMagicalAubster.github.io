AutoGrow


Description:
A hardware and software project to automate indoor vegetable growing.

Current status: In Progress
Note: this project is currently on hold due to some unfortunate plant diseases.

This project integrates a Raspberry Pi with sensors and pumps, using Python to control and deploy water as needed. The first iteration of this project is simply automatic watering.

The learning along the way for this project includes Python, hardware/software integration and calibration, and security. Future iterations of this project include adding and mixing plant nutrients with water, and addressing multiple types of plants.
